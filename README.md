# React Practice Projects

This repository contains multiple React projects that I built while learning and practicing React. The goal of this repository is to strengthen my understanding of React concepts by building practical applications.

## 🚀 Technologies & Concepts Covered

While working on these projects, I practiced and implemented:

* React Props
* React Hooks
* Context API
* Redux Toolkit
* React Router
* React Hook Form
* Tailwind CSS
* Local Storage
* Protected Routes
* Component-based architecture

## 📂 Projects Included

### 1. Todo List

A simple task management application where users can:

* Add tasks
* Delete tasks
* Mark tasks as completed
* Store tasks using Local Storage

### 2. Desi Swad Restaurant (Frontend)

A restaurant frontend UI built with React and Tailwind CSS featuring:

* Menu display
* Modern responsive layout
* Component-based UI

### 3. City Weather Search

A weather application where users can:

* Search weather by city
* View temperature and weather conditions
* Practice API integration and state management

### 4. Movie List App

A movie browsing application that:

* Displays movies in a list
* Uses protected routes
* Implements React Router for navigation

### 5. Mood Tracker

An application where users can track their mood daily.
It demonstrates:

* State management
* Local storage usage
* UI interaction

## 🎯 Purpose of This Repository

* Improve problem-solving skills
